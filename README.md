# WCAT
Udacity SPAIC - "Women code against terror" project

Scope of this project: Use deep learning to detect dangerous goods in x-ray scanned baggage.

Images included in the data folder are sourced from the "baggage" section of the following dataset: http://dmery.ing.puc.cl/index.php/material/gdxray/.
